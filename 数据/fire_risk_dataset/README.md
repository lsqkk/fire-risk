# ERA5数据集构建与处理

## 数据集选取与构成

采用欧洲中期天气预报中心（ECMWF）提供的ERA5再分析数据集，该数据集以其高时空分辨率和物理一致性在全球气象研究中得到广泛应用。针对山火预测任务的特点，我们精心选取了与火险密切相关的多维度气象变量，构建了一个综合性的气象数据集。

### 高空环流场数据

从ERA5气压层日统计数据集（derived-era5-pressure-levels-daily-statistics）中提取500hPa和850hPa两个关键等压面的气象要素，包括：

- 位势高度（geopotential）：反映大气环流形势

- 比湿（specific\_humidity）：表征大气水汽含量

- 温度（temperature）：热力条件指标

- U分量风场（u\_component\_of\_wind）：纬向环流特征

- V分量风场（v\_component\_of\_wind）：经向环流特征

### **近地面气象要素**

结合两个ERA5数据集获取地面关键参数。

#### 地表日统计数据集（derived-era5-land-daily-statistics）

- 2米露点温度（2m\_dewpoint\_temperature）：大气湿度指标

- 地表温度（skin\_temperature）：地表热力状况

- 10米U/V分量风场：近地面风场特征

- 地表气压（surface\_pressure）：气压场分布

- 高/低植被叶面积指数：植被覆盖状况

#### 单层数据集（reanalysis-era5-single-levels）

- 总降水量（total\_precipitation）：日累计降水

## 数据处理流程与方法

### 高效数据获取系统

本研究设计了基于CDS API与IDM（Internet Download Manager）的自动化数据获取系统。通过多线程并发请求技术，实现了对2024年3月至2025年2月期间我国东北地区（经纬度范围：124.00°E-128.08°E，43.00°N-47.40°N）数据的批量高效下载。该系统采用智能流量控制机制，确保在遵守CDS服务条款的同时最大化下载效率。

### 数据质量控制与标准化

原始数据经过严格的质量控制流程：

1. **完整性验证**：通过xarray库检查每个时间步的数据完整性，识别并标记缺失值
2. **物理合理性检验**：基于气象学原理验证变量数值范围合理性
3. **时空一致性分析**：确保相邻时空节点数据的连续性

### 多分辨率数据融合与插值

#### 降水数据特殊处理

考虑到降水场的强空间异质性和不连续性特征，本研究采用克里金插值法（Ordinary Kriging）进行处理。该方法基于地统计学原理，通过构建半变异函数模型捕捉降水场的空间自相关性，实现从0.25°原始分辨率到目标0.01°高分辨率的优化插值。

技术实现细节：

- 使用pykrige库实现普通克里金插值

- 采用球状变异函数模型拟合空间相关性

- 设置20个距离分箱优化计算效率

- 引入反距离权重法作为备选插值方案

#### 连续性气象场插值

对于温度、湿度、风场等连续性气象变量，采用xarray内置的双线性插值方法，该方法在保持物理场连续性的同时计算效率较高。

#### 空间裁切与网格标准化

为确保所有数据层空间一致性，采用精确的地理裁切技术将数据统一到研究区域（408×440网格点）。裁切过程中：

- 使用xarray的sel方法进行空间子集提取

- 保持纬度从北到南（47.40°N-43.00°N）的降序排列

- 确保经度从西到东（124.00°E-128.08°E）的升序排列

- 验证输出网格尺寸符合预期规格

### 多源数据集成

通过xarray的高级数据合并功能，将10个高空变量与8个地面变量整合为统一的多维数据集。集成过程中：

- 保持时间维度的一致性（2024年3月-2025年2月）

- 统一空间参考系统和坐标框架

- 保留原始数据的元数据信息和质量标志

## 技术栈与创新点

### 核心技术工具

- **数据处理**：xarray, NumPy - 多维数组操作和NetCDF格式处理
- **空间插值**：pykrige, SciPy - 高级插值算法实现
- **数据下载**：cdsapi, IDM - 高效数据获取管道
- **质量控制**：pandas, matplotlib - 数据验证和可视化

### 方法学创新

1. **多尺度插值策略**：针对不同物理特性的气象变量采用定制化插值方法
2. **自动化处理流水线**：实现从数据下载到最终产品的全流程自动化
3. **空间一致性保障**：通过统一的坐标系统和裁切标准确保多源数据匹配精度

### 数据集最终规格

经过系统处理后，构建的山火预测气象数据集具有以下特征：

-  **时间覆盖**：2024年3月1日-2025年2月28日（完整年度周期）

-  **空间范围**：东北地区124.00°E-128.08°E，43.00°N-47.40°N

-  **空间分辨率**：0.01°×0.01°（约1km）

-  **网格尺寸**：408（经向）×440（纬向）

-  **变量数量**：18个关键气象要素

-  **数据格式**：标准NetCDF4格式，支持高效存储和快速读取

该数据集为后续山火风险模型的训练和验证提供了高质量、一致性的输入数据，充分考虑了气象要素的物理特性和空间分布规律，为准确预测山火发生概率奠定了坚实基础。
